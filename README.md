# 📝 Advanced Todo App

A simple and powerful Todo List application built using **HTML, CSS, and JavaScript**.  
This project helps users manage daily tasks efficiently with features like edit, delete, filter, search, and local storage.

---

## 🔥 Features

- ➕ Add new tasks
- ❌ Delete tasks
- ✏️ Edit tasks
- ✔ Mark tasks as completed
- 💾 Save tasks in local storage
- 🔍 Search tasks
- 📌 Filter tasks (All / Completed / Pending)
- 🧹 Clear all tasks

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API
